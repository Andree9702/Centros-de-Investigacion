# Proyecto: Creación y Reglamentación de Centros de Investigación

## Objetivo
El objetivo principal de este repositorio es centralizar y gestionar la información, datos, análisis y documentación necesaria para la creación, formalización y reglamentación de los nuevos Centros de Investigación de la institución (inicialmente enfocados en FCA y FCQS).

## Alcance
Este proyecto abarca:
1.  **Recopilación y Limpieza de Datos:** Análisis de la carga académica, líneas de investigación y producción científica de los docentes.
2.  **Análisis Estadístico:** Evaluación de viabilidad, pertinencia y masa crítica para la formación de los centros.
3.  **Propuesta de Centros:** Elaboración de expedientes técnicos para la creación de los centros de investigación.
4.  **Marco Normativo:** Redacción y revisión del Reglamento General de Centros de Investigación y reglamentos específicos.

## Entregables
*   Base de datos consolidada y limpia de docentes y asignaturas.
*   Informes de análisis de pertinencia y viabilidad.
*   Expedientes de creación para el Centro de Investigación de la FCA y FCQS.
*   Borrador final del Reglamento General de Centros de Investigación.

## Estructura del Repositorio
*   `00_admin/`: Gestión del proyecto, hoja de ruta (roadmap) y bitácora de decisiones.
*   `01_datos/`: Datos crudos (`raw`) y procesados (`clean`), junto con el diccionario de datos.
*   `02_analisis/`: Notebooks de Jupyter, scripts de análisis y resultados generados.
*   `03_centros_propuestos/`: Documentación específica para cada facultad (FCA, FCQS, etc.).
*   `04_reglamentos/`: Borradores de normativas y plantillas legales.
*   `05_prompts_y_plantillas/`: Prompts para IA y plantillas reutilizables.

## Cómo Contribuir
1.  Asegúrese de leer el `roadmap.md` para entender las prioridades actuales.
2.  Para cambios en datos, utilice la carpeta `01_datos` manteniendo la distinción entre `raw` y `clean`.
3.  Para análisis, docuemnte sus notebooks en `02_analisis`.
4.  Mantenga el tono institucional y formal en toda la documentación.
